FinAnalytics is a robust Python-based desktop application designed to streamline personal expense management. By combining a user-friendly GUI with automated data categorization and visual analytics, it helps users transform raw spending data into actionable financial insights.

Key Features
Automated Categorization: Employs pattern-matching logic to instantly sort expenses (e.g., Food, Bills, Transport) based on transaction descriptions.

Persistent SQLite Storage: Utilizes a relational database backend to ensure data integrity and secure local storage.

Interactive Visualizations: Generates real-time Pie Charts and Bar Graphs using Matplotlib for spending distribution and trend analysis.

Modern GUI: Built with Tkinter, featuring a professional interface with input validation and instant feedback.

Tech Stack
Language: Python

GUI Framework: Tkinter

Database: SQLite3

Data Analysis: Pandas

Visualization: Matplotlib
